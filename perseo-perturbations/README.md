# PERSEO Perturbations

**P**ython **E**cosystem for **R**emote **S**ensing & **E**arth **O**bservation (PERSEO) package for computing **geodynamics displacements**
(Plate Tectonics, Solid Earth Tides) and **atmospheric delays** (Ionospheric, Tropospheric) needed for SAR
products analysis.

## Installation

This package can be installed using ``pip``:

```bash
pip install perseo-perturbations
```
