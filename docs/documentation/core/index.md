# CORE { #core data-toc-label="CORE" }

``Core`` is part of the `PERSEO` Aresys python project, and it's aimed at providing fundamental functionalities
for performing generic and broadly used operations, serving all other PERSEO modules and projects depending on PERSEO
framework.

The main functionalities provided are separated into thematic modules, each one providing a specific set of functions, classes
and methods. The two main modules implemented are:

<div class="result" markdown>

:lucide-shapes:{ .lg .middle } [__Geometry__](geometry/index.md#geom){ data-preview } : angles computation, coordinates conversion, attitude and trajectory management, geocoding features, etc.

:lucide-alarm-clock:{ .lg .middle } [__Timing__](timing/index.md#time){ data-preview } : precise timing for SAR operations, time conversion, etc.

:lucide-logs:{ .lg .middle } [__Logging__](logging/index.md#log){ data-preview } : useful logging features for PERSEO projects

</div>

!!! info "PERSEO Core"

    This package is the **foundation** of the `PERSEO` framework. It relies on ==battle-tested functionalities developed by the **Aresys team** throughout the years==, and it's the one that provides the most basic functionalities for performing generic and broadly used operations, serving all other PERSEO modules and projects depending on PERSEO framework.
