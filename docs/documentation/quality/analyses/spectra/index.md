# Spectral Analysis { #spectra data-toc-label="Spectral Analysis" }

Spectral Analysis can be used to investigate the spectral content of a selected data region by extracting absolute and/or
phase profiles in the frequency domain. 3 profiles are extracted both along range and azimuth directions at each third
of the data portion.

This analysis can be used on Point Targets or Distributed Targets data:

- **Point Target**: computing the absolute and phase spectra at each visible target location
- **Distributed**: computing the spectral amplitude on bursts or azimuth blocks of the input product
