# perseo-quality README

**P**ython **E**cosystem for **R**emote **S**ensing & **E**arth **O**bservation (PERSEO) package for SAR products quality
analysis and calibration assessment.

## Installation

This package can be installed using ``pip``:

```bash
pip install perseo-quality[graphs]
```

## Available functionalities

The PERSEO Quality package provides the following functionalities:

- Point Target Analysis
- Radiometric Analysis
- Interferometric Analysis
- Point & Distributed Target Ambiguity Ratio Analysis
- Point & Distributed Target Spectral Analysis
- Equivalent Number of Looks Analysis
- Elevation Notch Analysis

## Documentation

The PERSEO Quality package documentation can be found [here](http://intranet.aresys.it/sardashboard/develop/perseo/quality/docs/latest/).
